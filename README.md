# 🎮 Motion Capture Interactive System

這是一個結合電腦視覺與人機互動 (HCI) 的動作捕捉專案，透過辨識使用者的肢體動作，觸發對應的數位互動體驗。

## 🌟 專案亮點 (Features)
* 透過攝影機進行即時骨架/手勢追蹤
* 結合視覺回饋與音效互動 (Interactive Audio-Visuals)
* 探索無搖桿化 (Controller-free) 的自然使用者介面 (NUI)

## 💻 技術棧 (Tech Stack)
* Python
* Jupyter Notebook 
* 電腦視覺模型 (e.g., MediaPipe / OpenCV)
